# 手写高性能 GPT 实践
本仓库将从自注意力开始，然后实现一版基础的 GPT 模型, 接着在单张显卡上，实现计算性能优化，显存优化，最后实现分布式训练，最后实现高阶优化方法
本仓库将涉及到以下内容（踩坑）：
- GPT 代码实现
- 定位性能瓶颈的工具和方法
- 数据加载和预处理
- 单显卡性能优化，显存优化
- 分布式训练
- 高阶优化方法
